Для работы необходимо :

1. Создать БД и импотритовать дамп, который лежит в корне (parse_db.sql)
2. В config/db_params.php указать свои данные для подключения к БД.
3. В index.php установить нужный хост в константу HOST

По дефолту в config/cli_params.php установлена константа MAX_ITERATIONS_DEFAULT = 100. Это количество 
страниц, которое будет парситься, если при вводе команды парсинга не установить своё значение. 

В /config/all_params.php установлена директива xdebug.max_nesting_level = 5000. 
Для парсинга больших сайтов лучше поставить побольше.

Хэлп в консоли php help.php
